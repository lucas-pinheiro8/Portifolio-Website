NBA MVP Analysis (1981–2021)

Visão GeralProject Overview

Este projeto analisa 42 temporadas de votação do MVP da NBA para entender os fatores por trás do prêmio. O dashboard explora performance estatística, sucesso do time e comparações históricas entre os maiores jogadores da liga.

This project analyzes 42 seasons of NBA MVP voting to understand the factors behind the award. The dashboard explores statistical performance, team success, and historical comparisons between some of the greatest players in league history.

Dataset

Dados de votação do MVP da NBA cobrindo temporadas de 1981 a 2021, obtidos do Kaggle.

NBA MVP voting data covering seasons from 1981 to 2021 sourced from Kaggle.

Processo de DesenvolvimentoDevelopment Process

• Limpeza e transformação de dados com Power Query
• KPIs personalizados e cálculos avançados com DAX
• Análise estatística de médias de pontos, performance do time e padrões de votação
• Design de dashboards interativos com foco em storytelling visual

• Data cleaning and transformation using Power Query
• Custom KPIs and advanced calculations using DAX
• Statistical analysis of scoring averages, team performance and voting patterns
• Design of interactive visual storytelling dashboards
